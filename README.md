# 🏦 校园证券交易所

模拟校园生活中的股票交易，根据校园随机事件控制涨跌，以点数为资金。

## 🚀 快速运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务器
python main.py

# 3. 打开浏览器访问
http://localhost:8000
```

## ✨ 功能说明（MVP）

| 功能 | 说明 |
|------|------|
| 🔐 账户系统 | 注册/登录，每人1000点初始资金 |
| 📊 17只股票 | 17只社团股（热门社团 + 潜力社团） |
| 💹 实时交易 | 买入/卖出，支持1x/2x/3x/5x杠杆 |
| 📈 K线图 | ECharts 蜡烛图 + MA均线 + 成交量 |
| 📰 新闻事件 | 事件随机延迟2-15天后影响股价 |
| ⏱️ 时间推进 | 手动推进，共600教学日，跳过寒暑假 |
| 🛑 涨跌停 | ±15% 限制 |
| 🔄 强制平仓 | 杠杆仓位亏损超限自动平仓 |

## 📁 项目结构

```
模拟证券交易所/
├── main.py              # FastAPI 后端入口
├── game_engine.py       # 游戏引擎（时间/事件/价格）
├── database.py          # SQLite 数据库操作
├── stock_data.py        # 17只社团股定义
├── quiz.py              # 知识选择题题库
├── events.py            # 200+随机事件定义
├── requirements.txt     # Python 依赖
├── static/css/          # 样式文件
├── templates/           # HTML 页面
│   ├── login.html       # 登录/注册
│   ├── dashboard.html   # 主仪表盘
│   ├── trade.html       # 交易 + K线图
│   └── events.html      # 新闻事件
└── README.md
```
